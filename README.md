# 🐄 QurbaniHat - Digital Livestock Marketplace

QurbaniHat is a modern, responsive digital marketplace designed for buying and selling healthy, certified livestock for Qurbani. The platform connects users with the best breeds, ensuring a transparent and sacred booking experience.

## 🚀 Live Demo
**[Live Link](https://qurbanihat-livestock-booking-by-far-pi.vercel.app)**

## ✨ Key Features
*   **Modern Hero Section:** Visually appealing landing page with animations.
*   **User Authentication:** Secure sign-up, sign-in, and social login (Google) integration using Better-Auth.
*   **Livestock Catalog:** A dedicated "All Animals" section to browse available livestock.
*   **Responsive UI:** Optimized for all devices including mobile, tablet, and desktop.
*   **Personalized Profile:** Dedicated user profiles to manage bookings and personal information.
*   **Real-time Notifications:** Instant feedback for actions like login, logout, and sign-up using `react-hot-toast`.

## 🛠️ Tech Stack
*   **Frontend:** [Next.js](https://nextjs.org/) (React), [Tailwind CSS](https://tailwindcss.com/).
*   **UI Component Library:** [HeroUI](https://www.heroui.com/) (formerly NextUI).
*   **Authentication:** [Better-Auth](https://www.better-auth.com/).
*   **Icons:** [React Icons](https://react-icons.github.io/react-icons/) & Lucide React.
*   **Animations:** [Animate.css](https://animate.style/).
*   **Deployment:** [Vercel](https://vercel.com/).

## 🏁 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone (https://github.com/farhadahmdofficial/qurbanihat-livestock-booking-by-farhad/tree/main)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up Environment Variables:**
    Create a `.env.local` file and add your credentials:
    ```env
    BETTER_AUTH_SECRET=your_secret
    NEXT_PUBLIC_APP_URL=http://localhost:3000
    GOOGLE_CLIENT_ID=your_id
    GOOGLE_CLIENT_SECRET=your_secret
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
