# 🐄 QurbaniHat - Digital Livestock Marketplace

QurbaniHat is a modern, responsive digital marketplace designed for buying and selling healthy, certified livestock for Qurbani. The platform connects users with the best breeds, ensuring a transparent and sacred booking experience.

## 🚀 Live Demo
**[Live Link](https://qurbanihat-livestock-booking-by-far-pi.vercel.app)**

## ✨ Key Features
*   **Modern Hero Section:** Visually appealing landing page with high-quality animations.
*   **User Authentication:** Secure social login (Google) and credential-based access using Better-Auth.
*   **Dynamic Livestock Catalog:** Browse a detailed "All Animals" section with real-time data fetching.
*   **Interactive Booking System:** Seamless booking experience with instant UI feedback.
*   **Personalized Profile:** Dedicated dashboard to manage personal information and bookings.
*   **Smooth Animations:** Engaging user experience powered by Framer Motion and React Spring.

## 🛠️ Tech Stack

### Core Framework & Auth
*   **Framework:** [Next.js](https://nextjs.org/) (App Router)
*   **Authentication:** [Better-Auth](https://www.better-auth.com/)
*   **OAuth Provider:** [Google Cloud Console](https://console.cloud.google.com/)
*   **Database:** [MongoDB](https://www.mongodb.com/) (Data Persistence)

### UI & Styling
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Component Libraries:** [HeroUI](https://www.heroui.com/) & [Daisy UI](https://daisyui.com/)
*   **Design System:** [Gravity UI](https://gravity-ui.com/)

### Animations & Transitions
*   **Advanced Motion:** [Framer Motion](https://www.framer.com/motion/) & [React Spring](https://www.react-spring.dev/)
*   **Library Animations:** [Animate.css](https://animate.style/)
*   **Effects:** [React Fast Marquee](https://www.react-fast-marquee.com/) (For scrolling brands/announcements)

### Icons & Feedback
*   **Icons:** [React Icons](https://react-icons.github.io/react-icons/), [Iconify](https://iconify.design/), & Lucide React
*   **Notifications:** [React-Hot-Toast](https://react-hot-toast.com/) & [React Toastify](https://fkhadra.github.io/react-toastify/)
*   **Loaders:** [React Spinners](https://www.davidhu.io/react-spinners/) & [React Loader Spinner](https://mhnpd.github.io/react-loader-spinner/)

## 🏁 Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/farhadahmdofficial/qurbanihat-livestock-booking-by-farhad.git
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up Environment Variables:**
    Create a `.env.local` file:
    ```env
    BETTER_AUTH_SECRET=your_secret
    NEXT_PUBLIC_APP_URL=http://localhost:3000
    GOOGLE_CLIENT_ID=your_google_id
    GOOGLE_CLIENT_SECRET=your_google_secret
    MONGODB_URI=your_mongodb_connection_string
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
